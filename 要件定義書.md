## 1. プロジェクト概要

| 項目 | 内容 |
| --- | --- |
| プロジェクト名 | anju-portfolio.com リニューアルプロジェクト |
| 作成日 | 2025/10/31 |
| 作成者 | 安重 一弥 |
| 概要 | ポートフォリオサイトをリニューアルし、技術力・実績・人柄を効果的に伝える。 |
| 目的 | 案件獲得に繋がるブランディング強化を図る。 |

## 2. ゴール・目的

| 項目 | 内容 |
| --- | --- |
| 最終目標 | 企業やクライアントから仕事の相談が来るサイトにする。 |
| 成果指標 | 月間アクセス100以上、問い合わせ1件/月 |
| 想定ユーザー | Web制作会社の採用担当者、依頼を検討するクライアント |

## 3. ページ構成

| ページ名 | 目的 | 主な内容 |
| --- | --- | --- |
| トップページ（Home） | 第一印象・導線整理 | ロゴ（名前）・ナビゲーションメニュー・Instagramアイコン |
| 自己紹介（About） | プロフィール、経歴 |
| 制作実績（Works） | 使用技術・担当範囲 |
| ブログ（Blog） | 知識発信・学習記録 |
| お問い合わせ（Contact） | フォーム（メール送信・バリデーション） |
| 404 | エラーページ | ページが見つかりませんの表示・トップへ導線 |

## 4. 機能要件

| 機能名 | 内容 | 必須/任意 | 備考 |
| --- | --- | --- | --- |
| お問い合わせフォーム | メール送信（送信者・受信者） |
| ブログ管理（WordPress） |
| SEO対策 | タイトル・メタタグ・OGP設定 | 必須 | Google Analytics |

## 5. 非機能要件

| 項目 | 内容 |
| --- | --- |
| 表示速度 | Lighthouseスコア80以上を目標 |
| 技術構成 | HTML/CSS / jQuery / WordPress |
| 対応デバイス | PC / スマホ（レスポンシブ必須） |

## 6. デザイン要件
